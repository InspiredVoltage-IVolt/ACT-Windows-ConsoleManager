Template Structure

Here you can see the solutions of the template components that allows you to controll and to customise the appearance of your website. The main element is the header interactive icons using javascript info bubles and the most important is the horizontally aligned top menu.

Top Menu Description

The top menu is created by UL & LI tags designed by CSS only that allows mark active pages and hover items without using the javascript. The Javascript is used for the sliding out efect of subitems only and for the support of functionality for IE 6 browsers. The subitems are designed at the most to 4-th level.

Sidebar Description

Right menu displays static subitems of active page. If the page has no subpages the right menu will display the last level where you are. It means the right menu will be displayed in every time you browse the webpage.

You can type some text, links or other element to the grey box under the right menu and use it for the information displayed everywhere on the website. This place is suitable for simple icons or buttons to the most important products or services.

You can use the right column for the presentation of other services, too. Right bar can contains several static boxes like Product Presentation or other links displayed on every page.

Slideshows

The home page contains the great slider that use javascript efect to change the individual slides. These slides are simple pictures only. It allows you to utilise this space by any idea that will represent your business by using of simple promotion pictures.

Footer Boxes

The four boxes on the bottom of sub page are designed for the presentation of the most important services you provide. Every one of these boxes can be customised independently by many ways.
