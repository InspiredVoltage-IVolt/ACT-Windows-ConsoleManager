ACT Console Manager - Markup Language

The Markup Language was designed to allow you to create console user interfaces and process common actions without having to write any code. The best way to get started learning the markup language is to see an example. The first example below is an example of the default configuration markup file. This example show you how to use the basic markup settings.

Below is the list of available commands including a basic description. Clicking on the command will take you to the details page.
