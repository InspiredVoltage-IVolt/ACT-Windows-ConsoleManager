Overview

ACT Console Manager makes creating advanced windows console menus easy. The console engine supports the ACT Interface Plugin logic as well. You can incorporate both actions and authentication. Continue to read the docs and you will see and learn about the capabilities and purpose of this application.

Built in Editor

ACT Console Manager comes with default configuration files that allow you to quickly build a menu system without needing to code or change any files. The Editor allows you to add menu items in the console for testing and editing at the same time.

Custom Markup Language

The Console Manager comes with a custom console Markup language that allows you to create functional menus, designs, and other capabilities. Check out the full capabilities by looking at the Markup Language Topic. View Markup Language
